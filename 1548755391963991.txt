1. Разработать класс Image со следующим функционалом:
   - Загрузка\сохранение данных из файла png формата. (во время загрузки файл должен конвертироваться в ч\б изображение).
   - Функции доступа к данным изображения (пикселям, строкам, столбцам).

2. Написать тестовое приложение, использующее этот класс, которое будет:
  1..Принимать имя png файла из аргументов командой строки.
  2. Загружать файл в память.
  3. Подсчитывать количество пикселей, яркость которых превышает значение 0х80.
  4. Разбивать все изображение на 64 клетки (8 строк x 8 столбцов) и изменить яркость клеток в шахматном порядке. (как на шахматной доске).
  5. Полученное изображение сохранить в png файл под новым именем.

3. Перечислите использованные вами паттерны проектирования и обоснуйте их применение.

4. Код должен быть кросс платформенным и реализован на чистом С++ with stl, без использования сторонних библиотек.
Проект должен собираться cmake-ом.

Это задача без математики, достаточно простая, на проерку умения писать качественный код с++ и знание базовых паттернов.
Сроки не ставим, но чем быстрее тем лучше. 

Если хотите еще продемонстривовать свои базовые математические способности, то реализуйте любой алгоритм размытия изображения).
